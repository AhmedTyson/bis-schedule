# Project Constitution

This document defines the core principles and standards for the project.

## Project Vision

A fast, efficient, and user-friendly schedule management application.

## Development Principles

- **Spec-First**: Define features and technical plans before coding.
- **AI-Native**: Leverage AI assistants for generation, validation, and implementation.
- **Stability**: Ensure all changes are verified and don't break existing functionality.

## Tech Stack

- Frontend: HTML, CSS, JavaScript (Vanilla)
- Tools: Spec-Kit, pnpm, Netlify
